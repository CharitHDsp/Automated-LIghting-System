# Automated Lighting System

This automated lighting system can turn ON and turn OFF a light bulb at a specific time of day.
The job of providing the exact time is done by the RTC (Real Time Clock).

### Components list

- Arduino Uno Board
- DS3232 RTC Module
- Relay Module
- Light Bulbs
- 20x4 LCD Module
